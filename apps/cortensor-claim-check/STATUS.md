# PROJECT STATUS

**Date:** 2025-12-26

## Current Status
- Minimal fact-checking playground with Next.js UI and API
- Streams claims through a Cortensor router for decentralized fact-checking
- Aggregates verdicts, confidence, dispersion, reasoning, and miner votes
- Supports context via URL or explicit text (context_url/context_text)
- Miner cohort size configurable (default: 3, min: 1, max: 10)
- Uses Tavily API for evidence search and context hydration
- Displays aggregate and per-miner results, citations, and validation
- End-to-end polling, validation, and error handling
- Semantic versioning and changelog via RELEASE.md
